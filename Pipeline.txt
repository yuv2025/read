Practical Capstone — End-to-End Project Outline

Goal: Build and run the end-to-end pipeline once per day automatically.

Flow

Upload CSV to S3 (raw-bucket/YYYY/MM/DD/)

Airflow triggers Glue job (or a scheduled CloudWatch event)

Glue ETL (PySpark) reads CSV, transforms, writes partitioned Parquet to S3 and loads summary to RDS

Materialized views refreshed in Postgres (or refresh via SQL task in Airflow)

Streamlit dashboard reads from Postgres (cache) and shows MIS; users can export Excel

Automated email job attaches generated Excel reports and sends to recipient list

Airflow monitors success & sends alert on failures

Deliverables

Glue script (PySpark)

Airflow DAG to coordinate jobs

Streamlit app with authentication & export

CI/CD pipelines for code deploy

Monitoring + etl_runs logging + documentation
